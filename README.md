## 🇧🇷 Português
 
### Sobre o projeto
Calculadora de IMC desenvolvida como exercício prático para fixar os fundamentos de Java aprendidos nas primeiras aulas do curso.
 
O programa recebe a altura e o peso do usuário e calcula automaticamente o IMC, exibindo a classificação correspondente.
 
### Como usar
1. Execute o programa
2. Digite sua altura em metros (ex: 1.75)
3. Digite seu peso em kg (ex: 70.5)
4. O programa exibe seu IMC e a classificação
### Classificações
| IMC | Classificação |
|-----|--------------|
| ≤ 18.5 | Abaixo do peso |
| 18.5 – 24.9 | Peso ideal |
| 25.0 – 29.9 | Levemente acima do peso |
| 30.0 – 34.9 | Obesidade Grau I |
| 35.0 – 39.9 | Obesidade Grau II (Severa) |
| ≥ 40.0 | Obesidade Grau III (Mórbida) |
 
### O que aprendi desenvolvendo este projeto
- Estrutura condicional com `if / else if`
- Uso de operadores relacionais (`<=`, `>`, `<`) em intervalos
- Cálculo com expressões matemáticas (`peso / altura²`)
- Avaliar quando usar `switch` ou `if/else if`
### Tecnologias
- Java
- IntelliJ IDEA

## 🇺🇸 English
 
### About
A BMI calculator built as a hands-on exercise to reinforce Java fundamentals learned in the first lessons of the course.
 
The program takes the user's height and weight as input and automatically calculates the BMI, displaying the corresponding classification.
 
### How to use
1. Run the program
2. Enter your height in meters (e.g. 1.75)
3. Enter your weight in kg (e.g. 70.5)
4. The program displays your BMI and classification
### Classifications
| BMI | Classification |
|-----|---------------|
| ≤ 18.5 | Underweight |
| 18.5 – 24.9 | Normal weight |
| 25.0 – 29.9 | Slightly overweight |
| 30.0 – 34.9 | Obesity Grade I |
| 35.0 – 39.9 | Obesity Grade II (Severe) |
| ≥ 40.0 | Obesity Grade III (Morbid) |
 
### What I learned building this project
- Conditional structure with `if / else if`
- Relational operators (`<=`, `>`, `<`) for ranges
- Math expressions (`weight / height²`)
- Evaluating when to use `switch` vs `if/else if`
### Tech stack
- Java
- IntelliJ IDEA
